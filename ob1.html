Machine Learning y Deep Learning

  • Machine Learning es un conjunto de la técnica utilizada para el procesamiento de datos de gran tamaño mediante el desarrollo de algoritmos y un conjunto de reglas para entregar los resultados requeridos para el usuario.


  • Deep Learning representa un acercamiento más íntimo al modo de funcionamiento del sistema nervioso humano.
